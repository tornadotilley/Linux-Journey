`mv` is used for moving and renaming a file. 
- renaming the file - `[ttilley@somelinux ~] mv somefile.md otherfile.md`
- moving the file - `[ttilley@somelinux ~] mv somfile.md ~/somedirectory/`
- moving multiple files - `[ttilley@somelinux ~] mv somfile1.md somefile2.md ~/somedirectory/`
