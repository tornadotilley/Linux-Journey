- `*`  matches any characters
- `?`  matches any single character
![[Pasted image 20251207233605.png]]

- `*` matches all files
- `g*` matches any file beginning with 'g'
- `b*.txt` matches any file that starts with 'b' and ends with .txt
![[Pasted image 20251207233859.png]]

A wildcard can be combined with commands such as [[cp]], [[rm]], [[mv]], and many more.
